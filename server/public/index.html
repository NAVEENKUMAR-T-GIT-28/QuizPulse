QuizPulse API

Backend Services Online

The core backend engine handling real-time WebSocket communications, RESTful APIs, session management, and database operations for QuizPulse.

🔐

Authentication

POST /api/auth/register

Create a new host account

POST /api/auth/login

Authenticate and receive JWT token

GET /api/auth/me

Get current host profile (JWT required)

📝

Quiz Management

GET /api/quiz

List all quizzes for the authenticated host

POST /api/quiz

Create a new quiz with questions

PUT /api/quiz/:id

Update an existing quiz

🎮

Live Sessions

POST /api/quiz/:id/session

Launch live session & generate room code

GET /api/session/:roomCode

Validate room code for incoming players

GET /api/export/:sessionId

Export full session results as a PDF

📄

Export API

GET /api/export/:sessionId

Generate Mentimeter-style PDF analytics report.

GET /api/export/:sessionId/simple

Fallback lightweight PDF generation using PDFKit.

🗄️

Database Models

MODEL User

Stores host accounts, emails, JWT identity, and passwords.

MODEL Quiz

Stores quiz questions, answers, timers, and metadata.

MODEL Session

Stores responses, leaderboard scores, votes, and state.

🏗️

Architecture

CORE Express + Socket.io

Hybrid REST and WebSocket architecture for real-time multiplayer communication.

STATE Quiz State Machine

Server-controlled lifecycle: waiting → live → reveal → ended.

MEMORY Live Vote Cache

In-memory aggregation enables ultra-fast real-time analytics updates.

⚡

WebSocket Events (Socket.io)

Host Emissions

EMITquiz:start

Starts the session and broadcasts the first question

EMITquiz:reveal

Reveals correct answer & broadcasts computed scores

EMITquiz:next

Advances the game state to the next question

Player Emissions

EMITplayer:join

Connects to a live room using a display name

EMITplayer:answer

Submits a vote, instantly updating host analytics

EMITplayer:leave

Disconnects player from the active session
